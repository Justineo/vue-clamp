# vue-clamp

Vue 3 multiline text clamping powered by `@chenglou/pretext`.

## Development

- Check everything is ready:

```bash
vp run ready
```

- Run the tests:

```bash
vp test
```

- Run the browser-rendered DOM tests:

```bash
vp run test:browser
```

- Build the workspace:

```bash
vp run build -r
```

- Run the demo site:

```bash
vp run website#dev
```
