# 192 License and README refresh

## Goal

Restore the license text to match the historical `0.x` attribution and simplify the npm-facing
package README so it points users to the website for documentation.

## Steps

1. Compare the current license and README with the `0.x` branch.
2. Restore the MIT license text to the original attribution in both published license files.
3. Replace the package README with a simpler, website-first version.
4. Validate formatting and package contents.
