# Plan

## Goal

Use the `overlayscrollbars` package for the website page body and the website's actual scrollable
containers.

## Steps

1. Add `overlayscrollbars` to the website package and inspect the installed local typings so the
   integration matches the current package API.
2. Create a small website integration layer for the body and reusable element targets instead of
   scattering ad hoc setup calls through components.
3. Apply it to the body and the scrollable website containers, then update journey memory and run
   validation.
